# 🎯 Object Detection Using YOLO

A real-time object detection system built using YOLO (You Only Look Once) and OpenCV. The model can detect multiple objects in images and videos with high accuracy.

## 🚀 Features

- Real-time object detection
- Multiple object recognition
- Bounding box visualization
- High-speed inference
- Image and video support

## 🛠 Tech Stack

- Python
- YOLOv4
- OpenCV
- NumPy

## 📂 Project Structure

object-detection-yolo/
├── main.py

├── images/

## 📸 Output

Detected objects are highlighted with:
- Bounding boxes
- Confidence scores
- Class labels

## ▶️ Run Project

```bash
pip install -r requirements.txt
python main.py
```
